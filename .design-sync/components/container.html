Container

The content wrapper. Centres its children, caps them at --layout-content-width (70rem) and adds a --spacing-inset-m gutter so text never touches the viewport edge. Renders as div (default), main, header or footer.

The band is full-bleed; the dashed box is the container

max-width: var(--layout-content-width)  |  margin-inline: auto  |  padding-inline: var(--spacing-inset-m)

as="main" — the page's main landmark

Exactly one <main> per page. Header and footer use the same container so every band lines up on the same grid.
