Button

Two variants — primary (default) and ghost. Given an href it renders a real <a> that looks like a button.

variant="primary" — default

variant="ghost"

An icon-only button carries an aria-label — it must always have an accessible name.

disabled — 50% opacity, not-allowed cursor

href — renders an anchor styled as a button

An action that navigates (a mailto:, an external URL) must be a link, not a <button>, or it loses middle-click, "open in a new tab" and the right role. The href is handed to Link with variant="plain", so protocol handling — including the new-tab hint above — stays in one place.
